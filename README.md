# フリマ注文 → クリックポストCSV Chrome拡張機能

リベシティフリマの注文管理画面から、クリックポストの「まとめ申込」用CSVを一括生成するChrome拡張機能です。

## 機能

- 「取引中」タブの注文一覧にチェックボックスを注入
- 全選択 / 全解除トグル
- 最大40件のCSV一括生成
- 発送先情報（郵便番号・氏名・住所）の自動抽出
- 商品タイトルの自動抽出・整形
- Shift-JIS形式でCSVダウンロード
- 進捗表示付きローディングUI

## 対応サイト

- `https://furima.libecity.com/seller/orders`

## インストール方法

1. このリポジトリをクローン or ZIPダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `clickpost-csv-extension` フォルダを選択

## 使い方

1. リベシティフリマにログインした状態で注文管理ページを開く
2. 「取引中」タブを選択
3. 各注文行のチェックボックスで対象を選択（最大40件）
4. 「クリックポストCSV生成」ボタンをクリック
5. 自動的にCSVファイルがダウンロードされる

## CSVフォーマット

クリックポストの「まとめ申込」仕様に準拠：

| 列 | 内容 |
|---|---|
| A | お届け先郵便番号 |
| B | お届け先氏名 |
| C | お届け先敬称 |
| D | お届け先住所1行目 |
| E | お届け先住所2行目 |
| F | お届け先住所3行目 |
| G | お届け先住所4行目 |
| H | 内容品 |

- 文字コード: Shift-JIS
- 住所が20文字を超える場合は自動分割
- 氏名の「様」「御中」は除去（敬称列に移動）
- 商品タイトルは15文字以内に切り詰め

## 技術構成

- Manifest V3
- Content Script: 一覧ページUI注入
- Background Service Worker: fetch並列制御・CSV生成
- [encoding-japanese](https://github.com/nicktindall/encoding-japanese) v2: Shift-JIS変換

## 権限

- `activeTab`: アクティブタブへのアクセス
- `downloads`: CSVファイルのダウンロード
- `https://furima.libecity.com/*`: 注文ページへのfetchアクセス

## 注意事項

- サーバー負荷軽減のため、fetch は同時最大3件・500msインターバルで実行
- 取得失敗した注文はスキップされ、結果に表示される
- 実際のご利用前に少数件数でテストすることを推奨
